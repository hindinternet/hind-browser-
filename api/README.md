# API Module

This folder contains all API endpoints, request handlers, and communication logic for the Hind Browser project.

## Purpose of the API Layer

- Connect the frontend with backend services
- Validate incoming requests
- Handle authentication and permissions
- Fetch and send data from/to the database
- Provide modular and secure access to browser features

---

## Basic API Structure

1. **auth/**
   - Login / logout
   - Token generation
   - User session verification

2. **browser/**
   - Search requests
   - Tab management
   - History access

3. **web3/**
   - Domain resolver ( .hind )
   - Wallet connection
   - Blockchain-related API calls

4. **system/**
   - App info
   - Update checks
   - Config fetch
